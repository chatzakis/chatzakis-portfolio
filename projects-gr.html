Gym Rat Notes στο Διαδίκτυο 💪

Το Gym Rat Notes είναι η διαδικτυακή εφαρμογή που συνοδεύει την εφαρμογή Android Gym Rat. Αυτό το εύχρηστο σημειωματάριο γυμναστηρίου είναι εξοπλισμένο με μια ποικιλία από ασκήσεις και προγράμματα γυμναστικής, βοηθώντας τους χρήστες να παραμείνουν στους στόχους φυσικής κατάστασης τους. Το Gym Rat Notes λειτουργεί σωστά σε όλες τις συσκευές, παρέχοντας στους χρήστες μια σωστή εμπειρία σε υπολογιστές και σε φορητές συσκευές.

Ο έλεγχος ταυτότητας χρήστη εξασφαλίζει ασφαλή πρόσβαση, ενώ το frontend της εφαρμογής είναι κατασκευασμένο με χρήση React και Bootstrap για μια ομαλή, σύγχρονη διεπαφή. Το backend, που υποστηρίζεται από το Node.js και το PostgreSQL, συνδέεται με το frontend μέσω ενός REST API για αποτελεσματική διαχείριση δεδομένων σε πραγματικό χρόνο.

The X Quiz ❓

Το X Quiz είναι μια απλή εφαρμογή Node.js από την οποία οι παίκτες μπορούν να επιλέξουν διάφορες κατηγορίες και επίπεδα δυσκολίας και να ξεκινήσουν ένα γρήγορο κουίζ 10 ερωτήσεων. Οι ερωτήσεις επιλέγονται τυχαία κατόπιν αιτήματος στο TRIVIA API με βάση τις προτιμήσεις του παίκτη. Με την ολοκλήρωση του κουίζ, το παιχνίδι παρουσιάζει στον παίκτη την τελική του βαθμολογία, παρέχοντας μία ευχάριστη και εκπαιδευτική εμπειρία.

The Dilemma App 🆚 | MongoDB API

Μια απλή διαδικτυακή εφαρμογή που επιτρέπει στους χρήστες να δημιουργούν και να απαντούν σε διλήμματα (ερωτήσεις με δύο πιθανές απαντήσεις). Η εφαρμογή επιτρέπει στους χρήστες να επιλέξουν ανάμεσα σε δύο επιλογές, αυξάνοντας έναν μετρητή για κάθε επιλεγμένη απάντηση. Μετά την επιλογή, εμφανίζεται ποσοστό χρηστών που απάντησαν το ίδιο! Οι χρήστες μπορούν επίσης να υποβάλουν τις δικές τους ερωτήσεις στην βάση. Κατασκευάστηκε χρησιμοποιώντας Node.js, Express.js, και MongoDB. Η εφαρμογή περιλαμβάνει επίσης μια καθαρή διεπαφή χρήστη.

Διαδικτυακή πλατφόρμα δημιουργίας παιχνιδιών με κείμενο 🎮

Πρόκειται για ένα διασκεδαστικό έργο με σκοπό της δημιουργίας παιχνιδιών που βασίζονται σε κείμενο - αφήγηση, όπου οι επιλογές του χρήστη διαμορφώνουν το αποτέλεσμα της ιστορίας. Τα παιχνίδια τρέχουν στο πρόγραμμα περιήγησης σαν ιστοσελίδες και είναι φιλικά σε κινητές συσκευές

Η λογική του παιχνιδιού είναι φτιαγμένη σε JavaScript, ενώ η αφήγηση και οι ρυθμίσεις αποθηκεύονται σε ένα αρχείο JSON. Οι χρήστες μπορούν εύκολα να δημιουργήσουν τα δικά τους παιχνίδια τροποποιώντας το αρχείο JSON και προσαρμόζοντας τη CSS—δεν χρειάζεται να αγγίξουν τον κώδικα του παιχνιδιού.

Κατασκευαστής ιστοσελίδων ΙΚΕ

Αυτό το project επιτρέπει την δημιουργία μονοσέλιδων ιστοσελίδων HTML για χρήση από εταιρίες ΙΚΕ. Οι εταιρίες ΙΚΕ (Ιδιωτική κεφαλαιουχική εταιρεία) υποχρεούνται βάσει νόμου στην Ελλάδα να έχουν ιστοσελίδα με αναρτημένα τα στοιχεία της επιχείρησης. Χρησιμοποιώντας την σελίδα κατασκευαστής οι χρήστες μπορούν άμεσα δημιουργήσουν να κατεβάσουν την μία εταιρική ιστοσελίδα, βασισμένη σε ένα απλό πρότυπο, συμπληρώνωντας απλώς την φόρμα δημιουργίας.

Gym Rat Notes 📝

Ένα εύχρηστο σημειωματάριο ασκήσεων γυμναστηρίου για Android εμπνευσμένο από την ανάγκη παρακολούθησης της προόδου στο γυμναστήριο χρησιμοποιώντας smartphone.

    Επιλέξτε ανάμεσα σε προεγκατεστημένες ασκήσεις γυμναστικής ή δημιουργήστε τις δικές τους οργανωμένες σε μυικές ομάδες. Αναλύστε τις τελευταίες καταχωρήσεις σας για να σχεδιάσετε καλύτερα την επόμενη προπόνησή σας.

  2. Δημιουργία εξειδικευμένων προγραμμάτων

    Δημιουργήστε τα προσαρμοσμένα προγράμματα γυμναστικής σας συνδυάζοντας προεγκατεστημένες ή άλλες ασκήσεις!
